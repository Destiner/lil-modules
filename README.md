# lil modules

A collection of ERC-7579 modules.

> Unaudited, experimental, don't use in prod.

## Using this repo

To install the dependencies, run:

```bash
pnpm install
```

To build the project, run:

```bash
forge build
```

To run the tests, run:

```bash
forge test
```

- Validators

- Executors
